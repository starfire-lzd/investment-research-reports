# 规则候选池

本文件保存复盘中提出但尚未升级为正式规则的候选项。候选规则需要连续验证，不能因为单日表现直接写入 `rules.md` 或 `risk.md`。

## 升级标准

- 至少 3 个可验证样本。
- 成功率不低于 60%，或失败案例能明确帮助修正规则边界。
- 有清晰适用场景、触发信号、执行动作、风控条件和复盘要求。
- 不与现有风险约束冲突。
