# stock-pool - 自选股票池管理

## 使用场景

当用户需要：
- 添加股票到自选池
- 从自选池删除股票
- 查看当前自选股票列表
- 搜索自选池中的股票
- 管理股票分组（核心持仓、观察股等）

## 数据文件

**位置**: 工作空间根目录 `stock-pool.md`

**格式**:
```markdown
# 自选股票池

## 核心持仓
- 000001.SZ 平安银行
- 600519.SH 贵州茅台

## 观察股
- 300750.SZ 宁德时代
```

**格式规则**:
- `##` 作为分组标题
- 每只股票格式: `- {代码} {名称}`
- 代码格式: `{6位数字}.{SZ/SH}`

## 操作指南

### 添加股票

**命令示例**: `添加股票 000001.SZ` 或 `把 600519.SH 加到核心持仓`

**操作流程**:
1. 解析股票代码（支持自动补全交易所后缀）
2. 调用 `xtdata get_instrument_detail --stock-code "代码"` 获取股票名称
3. 若用户指定分组，添加到对应分组；否则询问或使用默认分组
4. 检查是否已存在，避免重复
5. 追加到 stock-pool.md

**代码补全规则**:
| 输入 | 补全为 |
|------|--------|
| `000001` | `000001.SZ` |
| `600000` | `600000.SH` |
| `300750` | `300750.SZ` |

**交易所判断规则**:
- `6` 开头 → `.SH`（上海）
- `0`、`3` 开头 → `.SZ`（深圳）
- `68` 开头 → `.SH`（科创板）

### 删除股票

**命令示例**: `删除 000001.SZ` 或 `把平安银行移出股票池`

**操作流程**:
1. 支持按代码或名称匹配
2. 展示匹配结果让用户确认
3. 从 stock-pool.md 移除对应行

### 查看列表

**命令示例**: `查看股票池` 或 `显示自选股`

**操作流程**:
1. 读取 stock-pool.md
2. 按分组格式化输出
3. 若文件不存在，提示股票池为空

### 搜索股票

**命令示例**: `搜索平安` 或 `查找 000001`

**操作流程**:
1. 按代码或名称模糊匹配
2. 返回匹配结果及所在分组

### 分组管理

**新建分组**: `新建分组 潜力股`
**重命名分组**: `重命名分组 观察股 为 待观察`
**删除分组**: `删除分组 待研究`（组内股票移至默认分组）

## 与 xtdata 联动

**获取股票名称**:
```bash
xtdata get_instrument_detail --stock-code "000001.SZ"
# 返回 InstrumentName 字段即为股票名称
```

**批量获取**:
```bash
xtdata get_instrument_detail_list --stock-list "['000001.SZ','600000.SH']"
```

## 注意事项

- 股票池数据存储在本地 stock-pool.md 文件中
- 添加股票时会自动通过 xtdata 获取股票名称
- 若 xtdata 服务不可用，提示用户手动输入名称
- 支持的交易所: SZ(深圳)、SH(上海)
